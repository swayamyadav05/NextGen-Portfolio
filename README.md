## Getting Started

First, run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the portfolio.

Access the Sanity Studio at [http://localhost:3000/studio](http://localhost:3000/studio) for content management.
